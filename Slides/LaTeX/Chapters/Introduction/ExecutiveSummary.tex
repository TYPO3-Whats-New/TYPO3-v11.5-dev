% TYPO3 Version 11.4 - Executive Summary

\begin{frame}[fragile]
	\frametitle{Einführung}
	\framesubtitle{Zusammenfassung}

	\small

		TYPO3 Version 11.5 ist das letzte Release der v11 Serie und erhält
		langfristigen Support (LTS). Diese Version wird für mindestens drei Jahre
		von der Community gepflegt.

		\vspace{0.2cm}

		Sie vereint viele Verbesserungen, neue Funktionen und moderne Technologien
		unter der Haube, die in früheren Sprint-Releases eingeführt wurden
		von Version 11.0 bis 11.4.

		\vspace{0.2cm}

%		We have hardly neglected an area to lift this enterprise CMS to a new
%		level. TYPO3 version 11 is for everyone and we can proudly claim that
%		this release is built to stay.
%
%		\vspace{0.2cm}

		\textbf{Ein herzliches und bescheidenes "Dankeschön!" an alle, die dazu beigetragen haben, dieses TYPO3 Release zu ermöglichen.}

		\vspace{0.2cm}

		Bitte beachten Sie, dass TYPO3 v11.5 auch zwei wichtige
		\href{https://typo3.org/help/security-advisories/typo3-cms}{Sicherheitsverbesserungen} enthält.
		Benutzer früherer v11-Versionen werden dringend gebeten, so bald wie möglich auf die neueste Version v11.5
		so schnell wie möglich zu aktualisieren.

	\normalsize

\end{frame}
